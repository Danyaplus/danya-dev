КАК ПОЛЬЗОВАТЬСЯ

1. Загрузи папку admin в корень репозитория:
   danya-dev/admin/index.html

2. Открой:
   https://danyaplus.github.io/danya-dev/admin/

3. Логин:
   danyaplus

4. Пароль:
   DanyaDev2026!

5. Заполни поля:
   RU — русский текст
   UA — украинский текст
   EN — английский текст

6. Нажми "Сгенерировать 3 карточки".

7. Вставь карточки так:

   RU-карточку → в index.html
   UA-карточку → в ua/index.html
   EN-карточку → в en/index.html

8. Вставлять нужно сразу после строки:

   <div class="blog-track" id="blogTrack">

   Тогда новая статья будет первой.
